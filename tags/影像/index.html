Zack Snyder's Justice League

整体观感非常好，路人超级英雄粉顺便也了解了一下当年院线版扑街的始末。很可惜，扎导这一部可以说是直接重启了DC大宇宙，大boss像是Darkseid，新成员如Cyborg，Flash和Martian Manhunter都被充实的有血有肉。甚至直接铺开了黑超的世界线，但是看情况短期不会再有第二部第三部了。

即便如此，还是有很多吐槽的地方：

  1. 蝙蝠侠露嘴的造型分分钟被打穿啊。但这一版本蝙蝠侠看起来精神了很多，大部分的打斗戏都不再羸弱，而且熟练运用最强钞能力，给整个小队成员提供技术支持。

  2. 人类保存MotherBox是不是也太寒碜了，直接找个地埋进去。史前大战这一幕很有魔戒的既视感，甚至战后分别保存Mother Box都和魔戒一样。

  3. 交通事故真的是万能cliche梗，即便如此只要运用得当也没问题。Cyborg的心路历程被交代的非常清晰，完全可以接着独立出电影了。

  4. 电影版BGM比扎导版好，特别是开头那一首。扎导选择的BGM太刻意了。

牯岭街少年杀人事件

我在台南，无聊的要命，每天可以看几十本武侠小说。后来，我叫他们帮我租最厚的武侠小说来看，其实以前的人跟我们现在出来混的人真的很像。有一个老包，大家都以为他吃错药，我记得，好像全城的人都翘头了，而且到处都被放火，他一个人要去堵拿破仑，后来还是被条子穴到了。其实以前的人，跟我们现在出来混的人，真的很像。《战争与和平》，其他的武侠书名都不记得了，只记得这一本。妈的，我还真想写本小说咧，干，给以后像我这种人看。太晚了，书又读的不够多。
